# The ActiveMe Way - Progress Tracker (prototype)

Interactive React prototype for ActiveMe 360's assessment logging and impact
reporting system. Fictional demo data only - not connected to any real school
or pupil data.

## Run locally

```
npm install
npm run dev
```

## Deploy

This is a static Vite app (`npm run build` outputs to `dist/`). Recommended:
push this repo to GitHub, then connect it to Netlify or Vercel for automatic
deploys on every push. See the accompanying deployment notes for step-by-step
instructions.
